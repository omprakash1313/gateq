retrieve_matrix(gate)

This function gets the given matrix after reading the inputs given by the user.

kr_product(inputs)

This function gives the Kronecker product of the given inputs. A Kronecker tensor product of A and B matrices is found by multiplying B with every element of A (commonly denoted as A⊗B). If A is an m-by-n matrix and B is a p-by-q matrix, then kron(A,B) is an m*p-by-n*q matrix.

Matrix_mult(x,y):

This function gives the matrix multiplication of the given inputs. This function is a primary requirement when finding final states, for example, by multiplying operator with 3-qubit basis state

Directions:


1) A list is initiated which stores each column of gates as a list.
2) Iterate through each list(column) in the initiated list.
3) retrieve_matrix retrieves the given matrix in the input, if a matrix is given with a
parameter, then it calculates and returns the resultant matrix.
4) After all the matrices are retrieved kr_product is used to perform Kronecker product of all the matrices present in each column and the resultant output is stored.
5) If the matrix in any column is of size 8*8, then that matrix is stored as is and other gates in the column are ignored. Note- Any 8*8 matrix should be placed in the first element of the column.
6) After the iteration is complete, Kronecker products of the column elements are stored.
7) Now function matrix_mult is used to perform matrix multiplication of all the resultant Kronecker products of the column elements.
8) Resultant output is operated on the initial state |000>.


Directions For usage:


1) The first thing displayed on the GUI is a table with 5 columns and 3 rows. The columns are meant for various gates to be employed to any of the 3 rows which represent the states.
2) Available gate options can be viewed from the repository on the right side of the screen. The repository is a list of gates and indicates the symbol for each gate.
3) Note that, a default ‘I’ can be observed written in each cell of the table. This is the identity matrix. To use a gate in a particular position, the ‘I’ is to be removed and the gate symbol needs to be typed in the above-mentioned table.
4) After the gates have been entered, the ‘SET VALUE’ button should be clicked to view the result of operations performed.
5) These results are printed on the screen in the form of a simple 8*1 matrix and also a dot graph for visualisation.
